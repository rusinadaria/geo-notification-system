## Тестовое задание на позицию Intern Golang Develop в Red Collar

### Инструкция по запуску 
1. Склонируйте репозиторий
2. Перейдите в корневую папку проекта
3. Соберите образы с помощью команды `docker-compose build`
4. Запустите контейнеры командой `docker-compose up`
5. Склонируйте репозиторий по ссылке: https://github.com/rusinadaria/news-portal и в корне проекта выполните команду `go run .`
#### --Настройка ngrok--
    5.1. Установите ngrok согласно инструкции на оф.сайте для вашей ОС https://ngrok.com/download/linux
    5.2. После установки, в открывшейся консоли выполните команду `ngrok http 9090`
6. Все готово!

*Для запуска юнит-тестов перейдите в директорию ./internal/handlers и воспользуйтесь командой `go test -v`*
