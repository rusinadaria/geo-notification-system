## Geo Notification API
Backend-сервис системы гео-оповещений. Разработан REST API, реализующий все CRUD-операции, с помощью которого оператор может создавать инциденты, сигнализирующие о каком-либо проиществии. Также реализован эндпоинт "/location/check", который проверяет очередь сообщений на наличие поступивших инцидетов. Если инцидент входит в заданный радиус - зону риска - на новостной портал (запускается как отдельный сервис https://github.com/rusinadaria/news-portal/blob/main/main.go) отправляется webhook с информацией об инциденте. Также есть эндпоинт "/system/health", который проверяет подключение к базам данных.

Используемые технологии: Golang, PostgreSQL, Redis, Docker

### Инструкция по запуску 
1. Склонируйте репозиторий
2. Перейдите в корневую папку проекта
3. Соберите образы с помощью команды `docker-compose build`
4. Запустите контейнеры командой `docker-compose up`
5. Склонируйте репозиторий по ссылке: https://github.com/rusinadaria/news-portal и в корне проекта выполните команду `go run .`
#### --Настройка ngrok--
    5.1. Установите ngrok согласно инструкции на оф.сайте для вашей ОС https://ngrok.com/download/linux
    5.2. После установки, в открывшейся консоли выполните команду `ngrok http 9090`
6. Все готово!

*Для запуска юнит-тестов перейдите в директорию ./internal/handlers и воспользуйтесь командой `go test -v`*
